# Realtime Web Traffic Monitor

![web-traffic](https://github.com/Dyuthiv05/web-traffic-analysis/blob/master/web-traffic.png)

## How to use

### Activate virtualenv
```
source bin/activate
```

### Install dependencies
```
pip install -r requirements.txt
```

### Run Flask app
```
cd src
python dbsetup.py
flask run
```
